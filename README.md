## Run Locally

Clone the project

```bash
  git clone https://github.com/jvgrini/feedback-modelling.git
```

Go to the project directory

```bash
  cd feedback-modelling
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Run a model

```bash
  python Models/M1/M1.py
```
